技术交底书是专利代理人理解发明点，写好申请文件的基础，应充分公开，使本领域普通技术人员能看懂。背景技术和详细技术方案部分一定要写得全面、清楚，不能仅仅针对本领域的技术专家，而忽略很多技术内容（即使发明人认为这些内容是业内都知晓的）。由于提交申请后不能对专利申请文件的公开内容进行修改，即不能改变、增加或删除，因此交底书必须充分完整。
技术术语应采用行业标准用语，全文对同一事物的叫法应统一，避免出现一种东西多种叫法。技术术语如果不是本领域技术人员常用和熟知的，尤其是自造词，应注明其定义。英文缩写后面用括号注明英文全称及中文解释。
专利代理人对交底技术内容承担保密责任，因此申请人应对代理人充分公开其发明创造技术内容。对于为了不让竞争对手完全掌握而希望保密的技术秘密，可以在细节上进行处理，如隐藏该部分技术内容或提供其他可行的实现方式。对于希望保密的技术内容，发明人需提前向专利代理人说明。

1、背景技术，即与本发明相关或相近的现有技术方案
请简介本发明应用领域，然后介绍相关的现有技术（如果现有技术出自专利、期刊、书籍，同时还要提供详细的出处）。现有技术方案可结合附图（如文字能描述清楚的，也可不用附图）进行清楚的文字描述，即别人是怎么做的；
指出现有技术方案中存在的缺点或不足（本发明不能解决的缺点，不需要指出），缺点可以根据本发明的优点来确定，可以是效率低、成本高、有污染、错误率高、反应速度慢等类似的问题，应当实事求是地写明缺点，即别人已经做了，我们为什么还要做类似的东西； 
应尽量全面，涵盖各种现有常见技术方案，然后指出各方案存在的缺点或不足。
2、本发明技术方案的详细阐述（发明内容）
2.1  本发明所要解决的技术问题（发明目的）
请针对现有技术的所有缺点，一一对应正面描述本发明所要解决的技术问题。
2.2  本发明提供的完整技术方案（发明方案）
1、技术方案部分为专利申请最重要的部分，需要详细提供；请仔细阅读本交底书的要求，材料思路一定要清晰，思路越清晰，内容越详尽，方案越完善，授权概率越高（一般要求文字部分在5、6页以上）。
2、专利必须是一个技术方案，应该阐述发明目的是通过什么样的技术方案来实现的，不能只有原理，也不能只做功能介绍。对于技术方案的描述，应当以本领域普通技术人员不需要再作任何设计、研究、改进或实验就能够实施，并且实施后能达到发明的技术效果为准。
3、请结合结构图、机械图、流程图、原理框图、电路图、时序图等附图进行说明，每个附图都应有对应的文字描述，以别人不看附图即可理解技术方案为准；请以黑白方式提供附图，不要提供彩色图例；各个附图中表示同一组成部分或同一部件的附图标记应当一致，说明书中和附图中使用的相同的附图标记应当表示同一组成部分或同一部件。
4、 描述发明内容时，不同类型的发明有不同的描述方式，例如：
产品发明：应当具体说明其零部件的结构及相互位置关系、连接关系和作用关系；
方法发明：应当说明为完成发明任务所必须实现的方法、流程和条件（如时间、压力、温度、浓度等）；说明书中如果出现公式，一定要给出公式中各个字母的含义，另外公式的推导一定要确保正确。
组合物或者混合物发明：应当说明其成分及含量（含量应说明是重量、体积或其它何种含量）。在描述其成分及含量时，尽可能提供其取值或者选择范围，并说明确定该范围的依据或者原因及技术效果。
电路发明：应当说明各功能电路之间的电连接关系或信号传送关系及各功能电路的具体构成；
软件程序发明：可站在产品经理的角度提供资料，包括提供详细的软件流程图，软件系统结构示意图，提供相关的系统装置，结合计算机硬件进行流程描述，写明软件执行的原理，而不需提供软件源代码（实用新型不涉及该类型）。
5、要写明本发明对现有技术的改进之处，即技术关键点；以及请求保护的发明点。
6、如果可能，尽量多写具体实施方式，实际实施的实验参数，产生的效果描述，尤其是有特点或特殊效果的实施例。
2.3  本发明技术方案带来的有益效果
1、结合技术方案来描述，做到有理有据；描述发明效果，能够量化的尽量定量描述，不能量化的，可以用对结构特点的分析和理论说明的方法加以描述，不要笼统的描述。
2、可以对应2.1部分所要解决的技术问题来描述。
2.4  是否还有其他替代方案
请分析是否还有别的替代技术方案同样能完成本发明目的；如果有，请尽量写明，内容的提供可以扩大专利的保护范围，防止他人绕过本发明去实现同样或类似的发明目的；“替代方案”可以是技术方案中部分结构、器件、方法步骤的替代，也可以是完整技术方案的替代。
3、附件：
是否已经发表过相关文章或论文、是否有参考文献（如专利/论文/标准）等?
若有，请说明出处，并提供相应的文本。另外，针对高校及科研院所，不仅仅要考虑个人发表过的论文，还应考虑本实验室的同学或同事所发表的相关论文。如已申请过类似专利也请一并提供。
提醒发明人注意：若相关内容之前确实已公开，请如实告知代理人，以便代理人在撰写时，针对已公开的内容规避处理；否则，已公开内容将影响本案的新颖性和/或创造性，使得本案授权概率极低，甚至为零。
